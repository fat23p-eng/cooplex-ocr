⚙️ วิธีตั้งค่าระบบ

1

ไปที่ Apps Script → Project Settings → Script Properties

2

เพิ่ม Property: GEMINI_API_KEY = API Key จาก aistudio.google.com (ฟรี)

3

สร้างโฟลเดอร์ใน Google Drive ชื่อ CoopLex_Laws แล้วอัปโหลดไฟล์ .txt กฎหมาย

4

เพิ่ม Property: LAW_FOLDER_ID = ID ของโฟลเดอร์ (จาก URL)

5

เพิ่ม Property: ADMIN_PASS, OFFICER_PASS, TOKEN_SECRET

6

กด Deploy → New Deployment → เลือก Web App → ตั้ง Execute as: Me และ Who has access: Anyone
